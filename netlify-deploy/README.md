# 🦷 **FEEL - Plateforme Dentaire**

## 🚀 **Déploiement Netlify**

Ce dossier contient tous les fichiers nécessaires pour déployer le site FEEL sur Netlify.

### **📁 Contenu du Dossier**
- `.next/` - Build de production Next.js
- `public/` - Assets statiques (images, logos)
- `package.json` - Dépendances du projet
- `netlify.toml` - Configuration Netlify
- `_redirects` - Gestion des routes
- `deploy-instructions.md` - Instructions détaillées

### **⚡ Déploiement Rapide**
1. **Glisser-déposer** ce dossier sur [netlify.com](https://netlify.com)
2. **Attendre** la fin du build (2-3 min)
3. **Partager** l'URL avec votre client !
